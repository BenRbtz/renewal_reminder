Using Renewal Reminder¶

renewal_reminder can be used in many ways:

  • invoked on the command-line

  • invoked via Python

  • invoked via Docker/docker-compose

Note

This portion of the documentation does not cover installation. See the Installation section for how to install the project.

Environment Variables¶

Before you consider running the application, refer to the below table for required/optional environment variables used:

100100100

Using Poetry¶

After setting the Environment Variables, use

# cd PROJECT_ROOT_DIR
make install
python renewal_reminder/app.py

Using Docker¶

After setting the Environment Variables, use

docker run -v $(PWD)/example.csv:/app/file_name.csv wondercipher/renewal_reminder

Note

The file path provided in environment variable APP_FILE_PATH, must be mounted in container. The mounted file and the environment variable filepath must match.
